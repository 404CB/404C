## 💎 为什么是 Claw-back？

说实话，这一轮 BNB Chain 上的 1% 税币已经泛滥。一开始大家都很兴奋——“有分红”、“有回购”、“有飞轮”。但最后呢？没人关心分红多少，因为价格已经不动了。

**问题不在机制，而在“没有刺激”。**

Claw-back 的出发点：如果一笔交易不能让人心跳加速，那它就没有意义。我们做的不是一个“分红币”，而是一个让每一笔交易都有可能改变命运的游戏。

## ⚡ 核心逻辑：CLAWBOT 在干什么？

你可以把 Clawbot 理解成一个一直盯着链上猎物的机器人。它不发工资，不讲感情，只执行以下三件事：

### 1. 🦞 CLAW GRAB（随机暴击）
**我们不做“平均主义”。**
每过一段时间，系统会突然触发一次“捕猎”——直接从最近的买家里抓一个人出来，将国库里一大块资金直接砸给他。没有预告，没有提示。你唯一能做的，就是在场。

### 2. 🔥 ALPHA SHIELD（价格护盘）
**防御不是为了好看，是为了让砸盘者付出代价。**
很多币死在同一个地方：有人砸盘导致恐慌，进而崩盘。Clawbot 实时盯盘，一旦跌得不对劲，直接动用资金回购并销毁。它在底部形成强力支撑，让市场知道：**这里不是随便能砸穿的地方。**

### 3. 👑 HODLERS' HONOR（长期奖励）
**市场最稀缺的资源不是钱，是耐心。**
Clawbot 会识别并记录从未卖出的地址。在未来的生态项目中，这些地址将获得绝对的优先权。不用喊口号，链上数据就是你的履历。
